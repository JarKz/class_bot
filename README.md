# class_bot - Простенькая программа для личного использования

Предназначена для моих личных целей, если хотите преобразовать его для своих целей, то нужно менять словари и расписание.


# Как запускать

Сначала установите такие фреймворки в виртуальную среду (например venv):
  - vkbottle
  - loguru (для отображения логов)
  - sqlalchemy
  - psycopg2

Создать в своей группе токен, и вставить его в cfg.py, чтобы он входил в кавычки.

Запустите через командную строку файл, в который поместили существующие на данный момент файлики здесь.

# Как пользоваться

Этот бот имеет набор простых команд, такие как запись домашнего задания в "дневник", отображение его, отображение их на завтра (или определённый день)

Больше информации получите при вводе "/помощь" в сообщениях боту.
